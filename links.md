# Photoshop drag and drop activation

Зайти в редактор реестра `Regedtit`

Перейти в папку:

```
HKEY_LOCAL_MACHINE
	SOFTWARE
		Microsoft
			Windows
				CurrentVersion
					Policies
						System
```

Выставить значчение параметра `EnableLUA` в `0`
